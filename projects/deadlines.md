# Deadlines

Live cross-project view of upcoming work, rendered by the Obsidian Tasks plugin (desktop only; on iOS use `TODO-widget.md`). Queries scope to `projects/`. Edit items in each `projects/<slug>/TODO.md` and the views below update automatically.

## Due in the next 7 days

```tasks
not done
folder includes projects
due before in 7 days
sort by due
sort by priority
hide created date
hide scheduled date
hide start date
hide recurrence rule
hide tags
hide backlinks
hide edit button
hide postpone button
hide toolbar
```

## All upcoming with a due date

Everything still open that has a 📅 attached, grouped by project and sorted by date.

```tasks
not done
folder includes projects
has due date
sort by due
group by folder
hide created date
hide scheduled date
hide start date
hide recurrence rule
hide tags
hide backlinks
hide edit button
hide postpone button
hide toolbar
```

## High priority without a due date

Items marked ⏫ or 🔺 that don't have a 📅 yet. Assign a date or downgrade.

```tasks
not done
folder includes projects
priority is above medium
no due date
sort by priority
hide created date
hide scheduled date
hide start date
hide recurrence rule
hide tags
hide backlinks
hide edit button
hide postpone button
hide toolbar
```
